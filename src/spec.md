# Specification

## Summary
**Goal:** Remove all non-vegetarian dishes from the recipe collection to display only vegetarian recipes.

**Planned changes:**
- Remove all non-vegetarian recipes (Butter Chicken, Chicken Biryani, and any other meat-based dishes) from the hardcoded recipe list in backend/main.mo
- Keep only vegetarian dishes in the recipe collection
- Frontend will automatically display the filtered vegetarian-only recipes without code changes

**User-visible outcome:** Users will see only vegetarian Indian recipes in the gallery, search results, and filtered views. All non-vegetarian dishes will be removed from the application.
